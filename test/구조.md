# Cafe24 제품 상세페이지 커스텀 테이블 분석 보고서

분석하신 사이트(`ecudemo387548.cafe24.com`)에서 "규격, 가격, 주문수량"이 포함된 테이블은 단순한 HTML 작성이 아닌, Cafe24의 **'관련상품(Related Products)' 모듈**을 커스텀하여 구현된 것입니다.

## 1. 구현 방식 상세

해당 부분은 Cafe24 편집창의 `product/detail.html` 파일 내에서 다음 모듈을 사용하여 구현되었습니다:

### 사용 모듈: `xans-product-relationlist`
일반적으로 "함께 구매하면 좋은 상품"을 보여주는 기능을 리스트 형태가 아닌 **테이블(Table) 형태**로 레이아웃을 수정한 구조입니다.

| 구분 | 구현 특징 |
| :--- | :--- |
| **데이터 소스** | 별도의 상품으로 등록된 각 규격별 제품들을 '관련상품'으로 연동 |
| **HTML 구조** | `<table>` 태그 내에 `<thead>`와 `<tbody>`를 사용하여 표 형식 구현 |
| **수량 조절** | Cafe24 기본 클래스인 `.RelationQuantityUp`, `.RelationQuantityDown` 사용 |
| **가격 처리** | 각 행의 체크박스를 선택하거나 수량을 변경하면 하단 "총 금액"에 실시간 합산 |

## 2. 주요 구성 요소 분석

### A. 테이블 헤더 (Thead)
```html
<thead>
    <tr>
        <th scope="col"></th>
        <th scope="col">규격</th>
        <th scope="col">가격</th>
        <th scope="col">주문수량</th>
        <th scope="col">비고</th>
    </tr>
</thead>
```
*   사용자가 보기 편하도록 표준 표 형식을 갖추고 있습니다.

### B. 반복되는 상품 행 (Tbody)
```html
<tr class="xans-record-">
    <td><input type="checkbox" id="related_check_0"></td>
    <td class="name">8.5T x 1220 x 2440 <span class="highlight">적라왕</span></td>
    <td class="price">70,000원/장</td>
    <td class="quantity">
        <input id="quantity_19" name="quantity_opt[]" value="10" type="text">
        <a href="#none" class="RelationQuantityUp"><img src="..."></a>
        <a href="#none" class="RelationQuantityDown"><img src="..."></a>
    </td>
    <td class="remarks">최소구매수량 : 10장</td>
</tr>
```
*   **상품명**: '규격' 열에 상품 이름을 출력합니다.
*   **수량 입력**: Cafe24의 `quantity_opt[]` 파라미터를 사용하여 장바구니에 한꺼번에 담을 수 있게 설계되었습니다.

### C. 하단 합계 영역
*   자바스크립트를 통해 상단 테이블에서 선택된 수량과 금액을 합산하여 보여줍니다.
*   "선택 수량 : X개 / 총 금액 : XXX,XXX원" 형태로 사용자에게 직관적인 정보를 제공합니다.

## 3. 이 방식의 장점

1.  **대량 구매 용이**: 사용자가 여러 옵션(규격)을 일일이 드롭다운에서 선택할 필요 없이, 한 눈에 보고 여러 개를 동시에 장바구니에 담을 수 있습니다.
2.  **재고 관리**: 각 행이 실제 개별 상품과 연결되어 있어, 규격별로 정교한 재고 관리가 가능합니다.
3.  **전문적인 UI**: B2B나 건축자재 쇼핑몰처럼 규격이 많고 정보가 중요한 사이트에서 신뢰감을 주는 디자인을 제공합니다.

---
> [!TIP]
> 만약 본인의 사이트에 이 기능을 직접 구현하고 싶으시다면, Cafe24 디자인 관리자에서 **[상품] > [상품상세정보] > [관련상품]** 설정에서 표시 형식을 수정하고 CSS로 표 스타일을 입히는 방식으로 접근하시면 됩니다.
