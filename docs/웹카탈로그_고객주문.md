# 웹카탈로그 고객주문 시스템 가이드

## 1. 이 시스템이 하는 일

고객이 웹카탈로그에서 상품을 담고
견적요청 버튼을 누르면
자동으로 구글시트에 기록되는 시스템입니다.

그림으로 설명:

고객
  ↓ 상품 담기 + 견적요청 버튼 클릭
웹카탈로그 (web-cadalog-ver10.vercel.app)
  ↓ 고객정보 + 상품목록 전송
Vercel 서버 (/api/submit-quote)
  ↓ Google Sheets API 호출
구글시트 (시트1) 자동 기록

## 2. 구글시트 출력 양식

| 열 | 내용 | 예시 |
|---|---|---|
| A열 | 고객 이름 | 홍길동 |
| B열 | 연락처 | 010-1234-5678 |
| C열 | 요청사항 | 빠른 배송 부탁드립니다 |
| F열 | 요청일자 | 2026-03-30 |
| H열 | 지점 | 안양 |
| J열 | 담당자 | 두현숙 |
| AA열 | 품목명 | 마대(80L) |
| AD열 | 수량 | 3 |
| AE열 | 가격 | 13500 |

## 3. 품목명이 나오는 원리

고객이 선택한 상품코드 (예: P0000CTY000A)
    ↓
[카페24상품] 시트 E열에서 찾기
    ↓
같은 행의 G열 값 = 이카운트 품목명
    ↓
구글시트 AA열에 출력

품목명은 **매 1시간마다 자동 최신화**됩니다.

## 4. 처음 설치 시 필요한 것

### 4-1. Google Cloud 서비스 계정

1. https://console.cloud.google.com 접속
2. API 라이브러리 → Google Sheets API → 사용 설정
3. 사용자 인증 정보 → 서비스 계정 만들기
   - 이름: quote-sheets-writer
   - 역할: 편집자
4. 키 탭 → JSON 키 다운로드
5. JSON 파일에서 아래 두 가지 값 복사
   - `client_email`
   - `private_key`

### 4-2. 구글시트 공유 설정

1. 출력 대상 구글시트 열기
   - ID: `1oQN0oApCGHSMHGYf_1gIpF-5dG8ETSsqrx-eAlz394k`
2. 공유 버튼 클릭
3. `client_email` 값 입력
4. 편집자 권한 부여

### 4-3. Vercel 환경변수 등록

Vercel → Settings → Environment Variables

- `GOOGLE_SERVICE_ACCOUNT_EMAIL` = client_email 값
- `GOOGLE_SERVICE_ACCOUNT_KEY` = private_key 값 전체
  - ⚠️ **주의: `\n`이 실제 줄바꿈으로 저장되어야 함**

등록 후 반드시 **Redeploy** 실행

## 6. 문제 발생 시 확인 순서

### Step 1: 견적요청 후 구글시트에 데이터 없음
- Vercel Logs에서 `/api/submit-quote` 500 오류 확인
- `GOOGLE_SERVICE_ACCOUNT_KEY` 환경변수 확인

### Step 2: AA열 품목명이 비어있음
- `[카페24상품]` 시트 G열 데이터 확인
- GAS `syncPrices()` 실행 후 G열 채워졌는지 확인

### Step 3: 견적요청 버튼 클릭 시 오류 팝업
- 브라우저 F12 → Network 탭
- `/api/submit-quote` 응답 코드 확인
