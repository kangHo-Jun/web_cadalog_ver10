# 웹카탈로그 고객주문(견적요청) 구현 가이드

## 1. 시스템 개요

기능: 웹카탈로그 견적요청 → 구글시트 자동 기록

전체 흐름도:
웹카탈로그 견적요청 버튼 → `/api/submit-quote`(Vercel) → Google Sheets API → 구글시트 `시트1` 기록

## 2. 전체 데이터 흐름

- 웹카탈로그 견적요청 버튼 클릭
- `/api/submit-quote`로 POST
- Google Sheets API로 스프레드시트 업데이트
- 구글시트 `시트1`에 행 추가

## 3. 필수 환경변수

- `GOOGLE_SERVICE_ACCOUNT_EMAIL`
- `GOOGLE_SERVICE_ACCOUNT_KEY`

설정 방법(요약):
- 서비스 계정 이메일과 private key를 Vercel 환경변수로 등록
- private key는 줄바꿈 처리 필요(아래 5장 참고)

## 4. Google Cloud 서비스 계정 설정 절차

1. Google Cloud Console 접속
2. 프로젝트 선택 또는 생성
3. APIs & Services → Library
4. Google Sheets API 활성화
5. IAM & Admin → Service Accounts → 서비스 계정 생성
6. 키 생성(JSON) 다운로드
7. 구글시트 공유
- 대상 스프레드시트를 서비스 계정 이메일로 공유(편집 권한)

## 5. Vercel 환경변수 등록 방법

- Vercel → Project → Settings → Environment Variables
- `GOOGLE_SERVICE_ACCOUNT_EMAIL`, `GOOGLE_SERVICE_ACCOUNT_KEY` 등록
- `GOOGLE_SERVICE_ACCOUNT_KEY` 줄바꿈 처리
- JSON 키의 `private_key` 값을 그대로 복사하면 `\n` 이 포함됨
- 코드에서 `replace(/\\n/g, '\n')` 처리 필요
- 환경변수 등록 후 반드시 Redeploy

## 6. 주요 파일

## 7. 구글시트 출력 구조

- 시트명: `시트1`
- C열: 요청일자
- E열: 안양
- G열: 두현숙
- X열: product_code
- AA열: 수량
- AB열: 가격
