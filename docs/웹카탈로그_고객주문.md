# 웹카탈로그 고객주문(견적요청) 구현 가이드

## 1. 시스템 개요

- 웹카탈로그 견적요청 → 구글시트 자동 기록
- 전체 흐름:
  웹카탈로그 → `/api/submit-quote` → Google Sheets API → 구글시트 `시트1`

## 2. 구글시트 출력 구조 (최종)

- A열: 고객 이름
- B열: 연락처
- C열: 요청사항
- F열: 요청일자
- H열: 안양
- J열: 두현숙
- AA열: 품목명 (PROD_DES — 이카운트 품목명)
- AD열: 수량
- AE열: 가격

## 3. PROD_DES 매핑 구조

- `[카페24상품]` 시트 G열 → PROD_DES 저장
- `syncPrices()` 실행 시 자동 업데이트 (1시간마다)
- variant_code → G열 PROD_DES 조회

## 4. Google Sheets API 설정

- Google Cloud 서비스 계정 생성
- Sheets API 활성화
- JSON 키 다운로드
- 구글시트 공유 (서비스 계정 이메일)
- Vercel 환경변수:
  - `GOOGLE_SERVICE_ACCOUNT_EMAIL`
  - `GOOGLE_SERVICE_ACCOUNT_KEY` (줄바꿈 `\n` 처리 필수)

## 5. 주요 파일

- `src/app/api/submit-quote/route.ts`
- `src/app/quote/summary/page.tsx`
- `scripts/p7_gas.js` (PROD_DES G열 저장 로직)
