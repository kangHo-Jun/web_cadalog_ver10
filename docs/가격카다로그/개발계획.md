### Phase 1-2 : 전일 대비 가격 변동 표시 ✅

**완료 내역**
- `/api/cron/snapshot-prices` 신규 생성
  - Vercel Cron 매일 KST 자정 실행 (UTC 15:00)
  - Redis 키 `price_snapshot:YYYY-MM-DD`, TTL 60일
- `/api/prices` 응답에 `previousPrice`, `changeAmount`, `changeDirection` 필드 추가
- 모바일 UI에 변동 표시 적용
  - ▲ 금액 +변동률% (빨강 #e53e3e)
  - ▼ 금액 -변동률% (파랑 #3182ce)
  - — 변동없음 (회색)
- ※ 첫 스냅샷 저장 후 24시간부터 변동 데이터 표시됨

## 2차 개발 (예정)

### Phase 2-1 : 카카오 로그인 연동

**목표**
협력업체 담당자만 가격정보 페이지 접근 가능하도록 인증 추가

**작업 내용**
- Kakao Developers 앱 등록 및 API 키 발급
- NextAuth.js 카카오 OAuth 구현
- `/login` 페이지 생성
- `/price` 페이지 — 미로그인 시 로그인 페이지로 리다이렉트
- (옵션) 허용된 카카오 계정만 접근 가능한 화이트리스트 기능

---

### Phase 2-2 : 한달 대비 가격 변동 표시

**목표**
전일 대비뿐 아니라 30일 전 대비 가격 변동률 추가 제공

**작업 내용**
- Redis에 30일치 가격 스냅샷 보관 로직 추가
- `price_snapshot:YYYY-MM-DD` 키 30일 TTL 설정
- 30일 전 스냅샷과 현재가 비교 → 변동률(%) 표시
- UI에 "전일 대비 / 30일 대비" 탭 또는 토글 추가

## 개발 원칙

- 기존 카탈로그 페이지는 절대 수정하지 않음 (롤백 보존)
- 새 기능은 항상 신규 파일/컴포넌트로 추가
- 환경변수 변경 시 반드시 Vercel 대시보드에서 확인
- 주요 작업 완료 후 `docs/` 폴더에 해결내용 기록
