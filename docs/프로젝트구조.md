## 🔌 GAS 프로젝트 구분

| 폴더 | 연결된 GAS 프로젝트 | 역할 | 트리거 |
| --- | --- | --- | --- |
| `gas-push/` | ERP_cafe24 | 이카운트 ↔ 카페24 가격 동기화 | syncPrices: 매 1시간 |
| `gas-push/` | ERP_cafe24 | 신규 상품 체크 | checkNewProducts: 매 2시간 |
| `gas-push/` | ERP_cafe24 | 카페24 캐시 갱신 | buildCafe24Cache: 매일 1회 |
| `gas-push/` | ERP_cafe24 | 토큰 만료 체크 | checkTokenExpiry: 매 1시간 |
| `monitoring-gas/` | 웹카다로그_모니터링_스크립트 | Redis/API/토큰 모니터링 대시보드 | checkAll: 매 1시간 |

## 🏗 주요 데이터 흐름

```
이카운트 API
    ↓ (가격 조회)
gas-push/Code.js (syncPrices)
    ↓ (가격 변동분만 PUT)
카페24 Admin API
    ↓
Redis (스냅샷 저장)
    ↓ (/api/debug-snapshot)
Phase1CatalogView (프론트)
    ↑ (모니터링)
monitoring-gas/Code.js (checkAll)
    ├── checkRedis        → /api/debug-snapshot (1회 공유)
    ├── checkSyncProducts → /api/debug-snapshot (1회 공유)
    └── checkCafe24Token  → /api/cron/check-token-expiry
```

## 🔑 clasp 명령어 참조

```bash
# 가격 동기화 스크립트
cd gas-push && clasp push    # GAS 배포
cd gas-push && clasp pull    # 로컬 동기화

# 모니터링 스크립트
cd monitoring-gas && clasp push   # GAS 배포
cd monitoring-gas && clasp pull   # 로컬 동기화
```
