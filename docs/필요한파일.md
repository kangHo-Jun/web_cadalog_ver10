# 프로젝트 필요한 파일 목록

아래는 프로젝트 전체 파일을 폴더별로 정리한 목록과 각 파일의 역할입니다.

## 루트
- implementation_plan.md: 구현 계획 정리 문서
- __init__.py: 루트 패키지 표시용 빈 모듈
- ecount_api_reference.md: 이카운트 API 참고 문서
- ecount_api_reference.md.gdoc: 이카운트 API 참고 문서(구글독 변환)
- cafe24_api.py: 카페24 API 연동 로직
- ecount_api.py: 이카운트 API 연동 로직
- ecount_api_test.py: 이카운트 API 호출 테스트 스크립트
- cafe24_Data.md: 카페24 데이터 구조 관련 메모
- download_products.py: 이카운트 품목 다운로드 실행 스크립트
- download_cafe24_sample.py: 카페24 샘플 데이터 다운로드 스크립트
- brute_force_api.py: API 호출 실험용 스크립트
- extract_plywood.py: 합판 관련 데이터 추출 스크립트
- find_product_api.py: 제품 조회 관련 API 호출 스크립트
- main.py: 메인 실행 진입점
- sync_engine.py: 동기화 엔진 로직
- verify_cafe24_auth.py: 카페24 인증 확인 스크립트
- verify_cafe24_step1.py: 카페24 연동 1단계 검증
- verify_erp_step1.py: ERP 연동 1단계 검증
- verify_transform_step2.py: 변환 단계 검증
- test_get_product.py: 제품 조회 테스트
- test_specific_api.py: 특정 API 테스트
- log.md: 작업 로그
- 이카운트_필드.md: 이카운트 필드 설명 문서
