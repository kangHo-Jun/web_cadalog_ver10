# Baseline v2 — 각재/목재

## UX 흐름
대표 상품 테이블 → 옵션 Glass Overlay → 견적 담기 흐름이 유지되어야 한다.

## 대표/옵션 매핑 기준
대표 상품과 옵션 상품은 동일한 상품 그룹 식별자(카페24 상품코드 prefix 또는 공통 그룹 키)를 기준으로 연결된다.

## Overlay 필수 요소
Glass 배경, 규격 선택, 수량 입력, ‘견적’ 문자열이 포함된 견적 담기 트리거가 포함되어야 한다.

## 성공 조건
각재/목재 기준으로 위 흐름과 요소가 충족되며 기존 기능·데이터·가격 결과에 영향이 없어야 한다.
