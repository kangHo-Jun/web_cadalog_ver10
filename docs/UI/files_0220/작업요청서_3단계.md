# 작업 요청서 — 3단계: 바텀 시트 컨테이너 내부 고정

> 우선순위: 보통 | 난이도: ★★★☆☆  
> 수정 파일: `BottomSheet.tsx` (또는 관련 시트 컴포넌트)  
> 선행 조건: 1단계 + 2단계 완료 후 진행

---

## 문제

바텀 시트가 `position: fixed` 기준으로 브라우저 전체 뷰포트에 붙어서  
900px 중앙 컨테이너를 무시하고 화면 전체를 덮는 현상.

## 요구사항

- 바텀 시트가 900px 컨테이너 안에서만 올라와야 함
- 배경 dimmed(어두운 반투명) 처리도 컨테이너 안에서만
- 기존 애니메이션(슬라이드 업) 유지
- 드래그 dismiss, 배경 클릭 닫기, X 버튼 닫기 모두 유지

## 수정 원칙

```
변경 전:  position: fixed  → 브라우저 뷰포트 기준
변경 후:  position: absolute → 부모 컨테이너(900px) 기준
```

> ⚠️ 2단계에서 부모 컨테이너에 `position: relative`가 적용되어 있어야 작동함  
> ⚠️ Overlay(배경 dimmed)도 동일하게 `absolute`로 변경할 것

## Framer Motion 설정 유지

```jsx
initial={{ y: "100%" }}
animate={{ y: 0 }}
exit={{ y: "100%" }}
transition={{ type: "spring", damping: 30, stiffness: 300 }}
drag="y"
dragConstraints={{ top: 0 }}
dragElastic={0.2}
onDragEnd={(event, info) => {
  if (info.offset.y > 100) onClose();
}}
```

## 완료 기준

- [ ] 바텀 시트가 900px 컨테이너 안에서만 나타남
- [ ] 배경 dimmed도 컨테이너 안에서만 적용됨
- [ ] 슬라이드 업/다운 애니메이션 정상 작동
- [ ] 드래그로 닫기 (100px 이상) 작동
- [ ] 배경 클릭 닫기 작동
- [ ] X 버튼 닫기 작동
- [ ] 시트 내부 스크롤 / 헤더 고정 정상 작동
