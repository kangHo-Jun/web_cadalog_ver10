# 작업 요청서 — 1단계: 스크롤 고정

> 우선순위: 긴급 | 난이도: ★☆☆☆☆  
> 수정 파일: `Phase1CatalogView.tsx` 1개만

---

## 문제

상품 전체 목록 로딩 시 페이지가 맨 아래까지 자동으로 내려가며 멈추는 현상.  
상품 데이터가 한꺼번에 렌더링되면서 페이지 전체 높이가 늘어나고 브라우저가 스크롤 위치를 잃는 것이 원인.

## 요구사항

- 브라우저 전체 스크롤 완전 차단
- 상품 리스트는 지정된 영역 안에서만 스크롤
- 초기 로딩 시 항상 맨 위에서 시작
- 기존 사이드바, 헤더, 카트바 구조 절대 건드리지 말 것

## 수정 원칙

`Phase1CatalogView.tsx` 안에서 메인 콘텐츠 영역 className만 수정.

```
전체 레이아웃:  h-screen overflow-hidden
메인 영역:      flex-1 overflow-hidden flex flex-col
리스트 영역:    flex-1 overflow-y-auto
```

## 완료 기준

- [ ] 전체 페이지 로딩 후 스크롤 위치가 항상 맨 위
- [ ] 상품 리스트가 리스트 영역 안에서만 스크롤됨
- [ ] 헤더, 사이드바, 하단 카트바 위치 변화 없음
- [ ] 기존 아코디언, 담기 버튼 동작 정상
