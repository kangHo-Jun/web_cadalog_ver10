## 작업 1: Redis 가격 스냅샷 API 생성

### `/api/cron/snapshot-prices/route.ts` 신규 생성
- 매일 자정 현재 Google Sheets 가격 전체를 Redis에 저장
- Redis 키: `price_snapshot:YYYY-MM-DD` (KST 기준)
- TTL: 60일
- 저장 구조: `{ [custom_variant_code]: price }` JSON

### `vercel.json` cron 등록
```json
{
  "crons": [
    {
      "path": "/api/cron/snapshot-prices",
      "schedule": "0 15 * * *"
    }
  ]
}
```
※ Vercel cron은 UTC 기준 → KST 자정 = UTC 15:00

### `/api/prices` 응답 수정
- 기존 응답에 아래 3개 필드 추가
  - `previousPrice`: 전일 스냅샷 가격 (없으면 null)
  - `changeAmount`: 현재가 - 전일가 (없으면 null)
  - `changeDirection`: `"up"` | `"down"` | `"same"` | `"none"`

## 작업 2: 모바일 전용 가격 조회 컴포넌트 생성

### `src/components/price/PriceCatalogMobileView.tsx` 신규 생성

**화면 구조**
┌─────────────────────────┐
│ 실시간 가격정보           │  ← 상단 헤더 (남색 #1e3a5f)
│ 🔍 검색창               │  ← sticky
├─────────────────────────┤
│ [전체] [단열재] [방수재]  │  ← 카테고리 칩 (가로 스크롤)
│ [미장재] [접착재] ...    │
├─────────────────────────┤
│ (카테고리 선택 전)        │
│ 카테고리를 선택해주세요    │  ← 안내 문구
├─────────────────────────┤
│ (카테고리 선택 후)        │
│ 품목명                   │
│ 규격           ₩12,500  │
│                ▲150 +1.2%│ ← 전일 대비 변동
│─────────────────────────│
│ 품목명                   │
│ 규격           ₩8,200   │
│                ▼300 -3.5%│
└─────────────────────────┘

**카테고리 칩**
- 상단 가로 스크롤 pill 형태
- 선택된 칩: 남색 배경 (#1e3a5f) 흰 글씨
- 미선택: 회색 테두리

**상품 리스트**
- 아코디언 없이 플랫 리스트 구조
- 장바구니/공유/담기 버튼 없음
- 버튼 없으므로 행 높이 최소화 → 빠른 스캔 가능

**가격 변동 표시**
- ▲ 금액 +변동률% → 빨강 (#e53e3e)
- ▼ 금액 -변동률% → 파랑 (#3182ce)
- — → 회색, 변동없음
- 전일 스냅샷 없을 경우 변동 표시 생략

**검색**
- debounce 200ms
- 품목명 기준 실시간 필터
