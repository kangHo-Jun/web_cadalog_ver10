# Cafe24 Web Catalog V10 UI Specification (2026-02-23)

이 문서는 현재 구축된 웹 카탈로그 V10의 UI 구조와 상호작용 로직을 정의합니다.

## 1. 레이아웃 구조 (Layout Architecture)

전체 화면은 900px 너비의 중앙 컨테이너 시스템을 유지하며, 각 컴포넌트는 다음의 포지셔닝 전략을 따릅니다.

| 컴포넌트 | 포지션 (Position) | 설명 |
| :--- | :--- | :--- |
| **StickySearchHeader** | `sticky top-0` | 스크롤 시 상단에 고정되는 검색 및 탐색 바 |
| **FixedSidebar** | `fixed` | 좌측 카테고리 메뉴 (메인 스크롤과 독립적) |
| **MainContent** | `relative` | 900px 내부의 실제 상품 리스트 영역 |
| **BottomCartBar** | `absolute` | 메인 컨텐츠 하단에 floating되는 장바구니 요약 바 |
| **CartDrawer** | `fixed` | 우측에서 슬라이드되는 전체 장바구니 상세창 |

## 2. 주요 컴포넌트 명세

### 2.1 하단 카트바 (BottomCartBar)
- **위치**: `main` 태그 내부 최하단 (`absolute bottom-0`)
- **스타일**:
    - 배경색: `#123628` (다크 그린)
    - 애니메이션: `framer-motion` (Spring Transition: y 100 → 0)
    - 주요 버튼: `Trash2` 아이콘 (비우기), `#48BB78` 배경 (견적요청)
- **상태 연동**: 장바구니 드로어가 열릴 때 시각적 간섭을 피하기 위해 `z-index`를 배경 오버레이 아래로 이동합니다.

### 2.2 장바구니 드로어 (CartDrawer)
- **위치**: 브라우저 우측 끝 (`fixed top-0 right-0`)
- **너비**: `360px`
- **백드롭**: `fixed inset-0` 배경 (`bg-black/30`, `backdrop-blur-sm`)

## 3. 상호작용 및 레이어 규칙 (Interactive Rules)

| 액션 | 결과 | 구현 세부사항 |
| :--- | :--- | :--- |
| **드로어 오픈 시** | 카트바 비활성화 | `opacity-30`, `pointer-events-none` |
| **레이어 순서** | 카트바 가리기 | `drawerOpen` 시 카트바 `z-index`를 `z-30`으로 하강 |
| **초기화 클릭** | 사용자 확인 | `window.confirm` 창 노출 후 장바구니 비우기 실행 |

## 4. 디자인 토큰 및 가이드

- **컨테이너 너비**: 최대 너비 900px 지향 (Main Content 영역)
- **CTA 컬러**: `#48BB78` (견적요청 등 주요 액션)
- **다크 컬러**: `#123628` (브랜드 아이덴티티 컬러)
- **폰트**: `next/font` 시스템 및 기본 프리미엄 타이포그래피 활용
