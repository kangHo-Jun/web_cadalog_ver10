# Cafe24 Web Catalog V10 UI Specification (2026-03-09 Update)

이 문서는 현재 구축된 웹 카탈로그 V10의 UI 구조와 상호작용 로직을 정의합니다.

## 1. 레이아웃 구조 (Layout Architecture)

기존의 `h-screen` 기반 내부 스크롤 방식에서 **전체 페이지 자연 스크롤(Natural Scroll)** 방식으로 전환되었습니다.

| 컴포넌트 | 포지션 (Position) | 설명 |
| :--- | :--- | :--- |
| **StickySearchHeader** | `sticky top-0` | 스크롤 시 상단에 고정되는 검색 및 탐색 바 |
| **FixedSidebar** | `block` (Flow) | 좌측 카테고리 메뉴. 메인 컨텐츠와 함께 자연스럽게 스크롤됨 |
| **MainContent** | `flex-1` | 전체 브라우저 스크롤을 따르는 상품 리스트 영역 |
| **BottomCartBar** | `absolute/fixed` | 장바구니 요약 바. 현재 `absolute`로 컨텐츠 최하단에 위치 (필요시 `fixed` 전환 가능) |
| **CartDrawer** | `fixed` | 우측에서 슬라이드되는 전체 장바구니 상세창 (백드롭 포함) |

## 2. 주요 컴포넌트 명세

### 2.1 하단 카트바 (BottomCartBar)
- **위치**: `main` 태그 내부 하단
- **VAT 표기**: 합계 금액 옆에 `(VAT 포함)` 명시
- **스타일**: 배경색 `#123628`, `framer-motion` 애니메이션 적용

### 2.2 상품 리스트 및 가격 표기
- **개별 상품/옵션**: 공급가(부가세 전)로 표시하며, 금액 옆에 `(VAT 별도)` 명시
- **장바구니 항목**: 공급가 합계 표시 및 `(VAT 별도)` 명시
- **최종 합계(Total)**: 부가세가 포함된 총액 표시 및 `(VAT 포함)` 명시

## 3. 상호작용 및 레이어 규칙 (Interactive Rules)

| 액션 | 결과 | 구현 세부사항 |
| :--- | :--- | :--- |
| **드로어 오픈 시** | 카트바 비활성화 | `opacity-30`, `pointer-events-none` |
| **레이어 순서** | 카트바 가리기 | `drawerOpen` 시 카트바 `z-index`를 `z-30`으로 하강 |
| **초기화 클릭** | 사용자 확인 | `window.confirm` 창 노출 후 장바구니 비우기 실행 |

## 4. 디자인 토큰 및 가이드

- **컨테이너 너비**: 최대 너비 900px 중앙 정렬 (`layout.tsx` 기준)
- **스크롤 시스템**: `globals.css`의 `height: 100%`, `overflow: hidden` 제거로 브라우저 기본 스크롤 활용
- **CTA 컬러**: `#48BB78` (견적요청, 담기 버튼)
- **다크 컬러**: `#123628` (헤더, 카트바 배경)
